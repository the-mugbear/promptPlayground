How to add a new Transformer

Follow these simple steps to complete the process to add and register your custom transformer.

  • 1 Step 1: Create a New Class

    Navigate to services -> transformers and use 'template.py' as a template. Copy the file and rename it to your desired transformer name. This will be used to bootstrap development of your custom/new transformer and help it conform to project standards.

  • 2 Step 2: Populate the Template

    Within this template, implement the transformation procedure and return the modified prompt. The template contains instructions highlighted below.

    id - Provide a concise string used by the application to reference the class
    name - Provide a name for the transformation
    description - Describe what the transformation does and how it acts on the input prompt

  • 3 Step 3: Modify registry.py

    Follow the conventions found within registry.py to accomplish the following:

    • Import your newly added transformer function at the top.
  • 4 Step 4: Add new feature to transformations partial

    Modify _transformations.html to include the newly created transformation. Reference existing transformation options. This partial is intended to consolidate transformation selection into one maintainable location.
