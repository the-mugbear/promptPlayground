How to Download and Use the Post Inspector Plugin

Follow these simple steps to download, install, and use the Post Inspector Plugin in your browser. Firefox support only.

  • 1 Step 1: Download the Plugin

    Click the 'Download Plugin' link below to get the packaged extension file. The file is named POSTInspector.xpi (for Firefox). Save this file to your computer.

    Download Plugin

  • 2 Step 2: Install the Plugin in Firefox

    Open Firefox and navigate to about:debugging#/runtime/this-firefox. Click on "Load Temporary Add-on" and select the downloaded POSTInspector.xpi file. The plugin icon should now appear in your browser's toolbar.

  • 3 Step 3: Use the PostInspector Plugin

    Click on the plugin's icon in your browser toolbar to open the Post Inspector popup.

    In the popup, you can view logged POST requests, filter them by hostname, and copy details such as Host, Filename, Request payload, and Request headers. You can also clear the logs when needed.
